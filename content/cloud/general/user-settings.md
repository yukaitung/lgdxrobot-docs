User settings are used to customise the user experience in LGDXRobot Cloud.

## User Details

* **Full Name**: The user’s full name.
* **Email Address**: The user’s email address.

## Password

To change the password, enter the current password, then enter the new password twice.

## Two-Factor Authentication (2FA) Settings

To enable 2FA, select **Setup Two-Factor Authentication**, follow the instructions, and save the recovery codes.

If the recovery codes are lost, select **Reset Recovery Codes**, then select **Start Reset Recovery Codes**.

To disable 2FA, select **Disable Two-Factor Authentication**, then select **Disable**.

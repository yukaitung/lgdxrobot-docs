LGDXRobot 2 MCU includs minimum source code, so it requires STM32CubeMX to generate entires project. You can choose any IDE to continue.

## Prerequest

Stm32cubemx
MDK-Community Edition or STM32 VS code extension

The project needed to be generated by STM32CubeMX, then launch the project using supported IDE.

## Configure

Open Inc/motor.h, update the constants according to your configuration.

```
// Constants - Chassis Configuration
#define CHASSIS_LX 0.237
#define CHASSIS_LY 0.287
#define WHEEL_RADIUS 0.0375
#define WHEEL_COUNT 4
#define MOTOR_GEAR_RATIO 90
#define MOTOR_MAX_SPEED float motor_max_speed[WHEEL_COUNT] = {10.948, 11.424, 11.1066, 10.6306}; // By testing

// Constants - Motor Configuration
#define ENCODER_PPR 3960

// Constants - PID Configuration
#define PID_KP float motor_kp[WHEEL_COUNT] = {6, 5, 6, 6};
#define PID_KI float motor_ki[WHEEL_COUNT] = {11, 11, 13, 13};
#define PID_KD float motor_kd[WHEEL_COUNT] = {0.0, 0.0, 0.0, 0.0};
```

## Next Step

You can test the chassis with ChassisTuner, which can test the speed of every motor and adjust the PID.

title: Controller Board (MCU)
layout: sub-section
weight: 2
---

The controller board is an STM32-based board designed specifically for the LGDXRobot2 platform. It supports a four-wheeled Mecanum chassis and can be adapted to other configurations. The source code is available in the [lgdxrobot2-mcu](https://gitlab.com/yukaitung/lgdxrobot2-mcu) repository.

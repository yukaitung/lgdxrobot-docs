title: Assembling the PC Plate
weight: 7
---

The installation procedure depends on the type of PC being used. Three types of PC are supported: **Nvidia Jetson Nano**, **Raspberry Pi**, and **NUC**.

Insert M4 screws through the PC plate into the M4 standoffs and tighten them with a hex key. Then install the PC according to its type.

![Image of the install standoffs](../img/asm_pc/standoffs.png)

### Nvidia Jetson Nano

Insert M2.5 screws through the PC plate into the M2.5 standoffs and tighten them with a hex key. 

![Image of the install standoffs](../img/asm_pc/jetson_standoffs.png)

Secure the Jetson Nano to the standoffs using M2.5 screws and a hex key.

![Image of the install Jetson Nano](../img/asm_pc/jetson_install.png)

Insert M4 screws through the PC plate into the M4 standoffs and tighten them with a hex key. Then secure the PC plate to the bottom plate using M4 screws and a hex key. Connect the USB cables for the controller board, camera, and LiDAR.

![Image of the install on chassis](../img/asm_pc/jetson_chassis.png)

### Raspberry Pi

Insert M2.5 screws through the PC plate into the M2.5 standoffs and tighten them with a hex key. 

![Image of the install standoffs](../img/asm_pc/pi_standoffs.png)

Secure the Raspberry Pi to the standoffs using M2.5 screws and a hex key.

![Image of the install Raspberry Pi](../img/asm_pc/pi_install.png)

Insert M4 screws through the PC plate into the M4 standoffs and tighten them with a hex key. Then secure the PC plate to the bottom plate using M4 screws and a hex key. Connect the USB cables for the controller board, camera, and LiDAR.

![Image of the install on chassis](../img/asm_pc/pi_chassis.png)

### NUC

The case of the NUC must be removed before installing on the PC plate.
{.alert .alert-warning}

Place the NUC on the PC plate and secure it using M3 screws from the underside of the NUC, tightening with a hex key.

![Image of the install NUC](../img/asm_pc/nuc_install.png)

Secure the PC plate to the bottom plate using M4 screws and a hex key. Connect the USB cables for the controller board, camera, and LiDAR.

![Image of the install on chassis](../img/asm_pc/nuc_chassis.png)

The controller board is an STM32-based controller board designed specifically for the LGDXRobot2 platform. It supports a four-wheeled Mecanum chassis and can be adapred to other configurations. The source code can be found in the [lgdxrobot2-mcu](https://gitlab.com/yukaitung/lgdxrobot2-mcu) repository.

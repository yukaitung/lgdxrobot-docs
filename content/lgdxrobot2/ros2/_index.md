LGDXRobot2 ROS 2 is an integration layer for the ROS 2 ecosystem, with examples for both physical robots and simulations. It is developed in ROS 2 Jazzy on Ubuntu 24.04 and offers seamless integration with the [LGDXRobot Cloud](https://gitlab.com/lgdxrobotics/lgdxrobot2-cloud).

The source code is available in the **LGDXRobot2 ROS 2** repository ([GitLab](https://gitlab.com/lgdxrobotics/lgdxrobot2-ros2) | [GitHub](https://github.com/yukaitung/lgdxrobot2-ros2)).

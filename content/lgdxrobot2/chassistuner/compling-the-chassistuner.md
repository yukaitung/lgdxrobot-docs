The ChassisTuner is not currently distributed as a binary.

## Prerequisites

Qt 6.10, including the Qt Serial Port and Qt Graphs modules. Install it using the Qt Installer.

## Build

1. Clone the [lgdxrobot2-chassistuner](https://gitlab.com/yukaitung/lgdxrobot2-chassistuner) repository.
2. Open the project in Qt Creator.
3. Press **Run** to build and launch the application.
